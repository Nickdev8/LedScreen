To update an animation. it should be as easy as putting the new animations on the sd card.

The file you want to put on there is generated by XLights. the project is linked in ../xlights_project/ here you should set the panel size and make your animation. To see a live preview. simplily plug the board in via the USB port.

To change the board size, you need to update the code. please refer to the [config file](../arduino_firmware/RPI-PICO/Config.h) and [firmware inctructions](FIRMWARE_INSTUCTIONS.md)
